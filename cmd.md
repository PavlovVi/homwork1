# Подсказка по командной строке

## 1 создать файл

```sh
touch NameFile.XXX
```

## 2 сменить директорию

```sh
cd
C:\path\to\folder
```

## 3 очистить терминал Ctrl + L

```sh
cls
```

## 4 листинг файлов

```sh
dir
```

## 5 редактор в консоли

```sh
vim
```

## посмотреть журнал

```sh
git log
```

## создать директорию

```sh
mkdir FolderName
```
